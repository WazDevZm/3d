Features

Everything in the browser

Built with open-source tools. Works on any modern laptop with a webcam.

✋

Real-time hand tracking

MediaPipe detects 21 landmarks on each hand at up to 30 fps. Both hands tracked simultaneously.

🔌

Live circuit simulation

Graph-based DFS engine checks for completed circuits. LEDs light up the moment a loop closes.

📦

3D component workspace

Three.js renders resistors, LEDs, batteries, and breadboards with pickup shadows and tilt physics.

🖐️

Pinch gesture control

Hysteresis-based pinch detection prevents jitter. Grab, drag, and drop with natural hand motion.

🔗

Wire drawing mode

Pinch any terminal to start drawing a wire. Release on another terminal to make a connection.

🎒

Works on any device

No downloads, no build step, no accounts. Open the URL on any computer with a webcam.

How it works

Three steps to your first circuit

01

Allow camera access

Click Allow when the browser prompts for webcam permission. Video is processed entirely in your browser — nothing is ever sent to a server.

02

Pick up components

Hold your hand over a component and pinch your thumb and index finger together to grab it. Move your hand to drag it across the workspace.

03

Connect and complete

Switch to Wire mode, pinch a terminal, drag to another terminal, and release. When the circuit closes, the LED lights up.

Gesture reference

Controls at a glance

🤌

Pinch — grab / release

Bring thumb and index finger together to grab. Open to release.

🖐️

Open hand — hover

Move an open hand over components to highlight them.

🔄

Two hands — rotate

Grab with one hand, use the other to rotate the component around its axis.

👆

Pinch terminal — wire

In Wire mode, pinch a terminal point to start drawing a connection.

Tech stack

What this is built with

Every piece is open-source and runs entirely in the browser.

🎨
Three.js r160
WebGL 3D rendering — geometry, lighting, materials, animation loop
👁️
MediaPipe Hands
21-landmark real-time hand detection via a TFLite WASM model
📦
ES Modules + importmap
Native browser modules with no bundler, no build step required
🔢
Graph DFS simulation
Adjacency map circuit engine; finds battery→LED loops in real time
🐍
Python HTTP server
Minimal dev server that sets correct CORS and MIME headers
🎛️
Vanilla JS
Zero frameworks — gesture math, coordinate mapping, lerp animation
